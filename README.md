# Full-Stack-Development-3

Composants : Les composants sont les éléments fondamentaux de React, et ils permettent de diviser l'interface utilisateur en morceaux réutilisables et autonomes. Un composant peut être une petite partie d'une page ou une entité plus complexe qui contient d'autres composants. Ils encapsulent la logique, la structure et le style liés à une partie spécifique de l'interface utilisateur.

Props : Les props, abréviation de propriétés, sont un mécanisme essentiel de communication entre les composants dans React. Ils sont utilisés pour transmettre des données spécifiques d'un composant parent à un composant enfant. Les props sont immuables, ce qui signifie qu'ils ne peuvent pas être modifiés à l'intérieur du composant qui les reçoit. Ils permettent de personnaliser le comportement d'un composant en fonction des valeurs fournies par son parent.

État : L'état, ou state en anglais, est un concept clé dans React pour gérer les données qui peuvent changer au fil du temps. Contrairement aux props, l'état est géré à l'intérieur d'un composant et peut être modifié par ce dernier. Lorsque l'état d'un composant est mis à jour, React se charge de mettre à jour automatiquement l'interface utilisateur pour refléter ces changements. L'état permet de créer des composants interactifs et dynamiques, où les modifications de données déclenchent des mises à jour de l'interface en temps réel.
